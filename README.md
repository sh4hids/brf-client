A react starter kit with opinionated project structure.

## Uses

- Install dependencies

```bash
$ npm i
```

- Run the project

```bash
$ npm start
```

- Build the project

```bash
$ npm run build
```

- Test the project

```bash
$ npm run test
```

- Configure
  Please rename the file `./src/config.example.js` to `./src/config.js` and change the configuration accordingly.
